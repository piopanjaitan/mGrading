# Dokumentasi Tes Lapangan mGradingUSTP

Folder ini berisi hasil dokumentasi tes lapangan dari data pada `Live_testing_documentation`.

## Laporan Utama

- `mGradingUSTP_laporan_tes_lapangan.md`
- `mGradingUSTP_laporan_tes_lapangan.html`
- `mGradingUSTP_laporan_tes_lapangan.pdf`

## Asset Pendukung

- `assets/captures/` - capture awal, tengah, akhir, dan contact sheet video.
- `assets/charts/` - grafik distribusi label, confidence, session, dan alur data.
- `assets/images/` - grid contoh crop dan frame dari app.
- `assets/tables/` - CSV ringkasan video, label, session, dan top confidence.
- `source_manifest.json` - ringkasan sumber dan output yang dihasilkan.

Ringkasan data: 499 record TBS, 8 live session, 6 video MP4, 4 video unik, 281 frame, 449 crop, dan 0 file annotated pada paket lapangan ini.
